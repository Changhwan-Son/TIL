### Vue.js

- MVVM 패턴
  - Model + View + ViewModel
    - Model : JS 객체
    - View : 화면 DOM
    - ViewModel : Vue의 역할. View와 Model 사이 데이터 바인딩 -> 양방향 통신 가능하게 함
- Vue Instance

  ```
  // 인스턴스
  new Vue({

  });
  ```

  - 속성
    - el
    - data
    - methods
      ...
  - 과정
    뷰 라이브러리 파일 로딩 -> 인스턴스 객체 생성 -> 특정 화면 요소에 인스턴스 붙임 -> 인스턴스 내용이 화면 요소로 변환 -> 변환된 화면 요소를 사용자가 최종 확인

- Vue Life Cycle
- Vue Directive(디렉티브)
  - v-model
  - v-bind
  - v-if, v-else-if, v-else
  - v-show
  - v-for
